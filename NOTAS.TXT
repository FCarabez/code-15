// Terminal Linux
// Comandos de Linux
// Windows -> Powershell
No podemos movernos con un click, usamos flechas del teclado
Una línea de código o un comando nos permite ejecutar una acción

Comandos
pwd (print working directory)-> me muestra en que directorio o ubicación de mi computadora me encuentro
Ejemplo: Users\Pacoc
ls (list)-> me permite ver los elementos que hay en mi ubicación actual
cd (change directory) -> me permite ingrear a una carpeta del directorio que tengo disponible
